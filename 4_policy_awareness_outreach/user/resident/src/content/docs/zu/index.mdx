Izikhukhula ziwusongo olukhulu ezindaweni ezahlukahlukene, zenzeka phakathi kwezinyanga zonyaka. Imvamisa, izikhukhula zenzeka ngenxa yemvula eqhubekayo ezinsukwini eziningi, imvula enamandla esikhathini esifushane, noma ngenxa yokuphakama kweqhwa noma imfucumfucu okwenza umfula noma umfudlana ugcwale. Zingase futhi zibe ngenxa yokwehluleka kwesakhiwo sokulawula amanzi, njenge-levee noma idamu. Isizathu esivamile sokukhukhula kubangelwa ukungakwazi komhlabathi ukumunca amanzi ngesikhathi esifushane, noma uma imifula ingakwazi ukuhamba. Cishe amaphesenti angamashumi ayisikhombisa nanhlanu wezinhlekelele ezimemezelwa ahlobene nezikhukhula. Ngezansi kukhona izingozi ezithile ezivame ukuba khona ngezikhukhula.
